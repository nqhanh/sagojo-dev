The Tabber-Widget easily allows you to add multiple tabbed widgets in to one widget area.

== Description ==

Use the Tabber widget to add up to 10 tabbed widget areas in to one existing widget.

Tabber Widget works in all browsers, and comes packed with 4 templates that can be customized to the look of your site.

== Installation ==

1. Download and unzip the latest release zip file
2. If you use the WordPress plugin uploader to install this plugin skip to step 4.
3. Upload the entire plugin directory to your `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress Administration

* Usage:
1. Navigate to the new menu for this plugin in the Wordpress Administration Panel under Tools -> Tabber Widgets
2. Create a new Tabber by giving it a name and selecting the number of tabs you would like.
3. Navigate to the Widgets menu in the WordPress Dashboard under Appearance -> Widgets
4. From the Available Widgets column, drag the Tabber Widget to the disred widget area.
5. Give the tabbed area a widget title (optional) and select the appropraite Tabber from the dropdown menu.
6. On the right hand side of the menu, fill the new "Tabber" widgets with the widgets you would like to appear in each tab.
7. Rinse and repeat.
